### python 설정 (mac os 기준)
```
# 1. python.org에서 버전 설치
> which python
> echo ~

# 2. pip 버전 설정
> which pip3    # /usr/local/bin/pip3
> alias pip=/usr/local/bin/pip3
> pip3 install --upgrade pip
```
